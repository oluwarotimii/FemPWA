## 🔧 Deployment Steps

### Step 1: Build Locally

```bash
cd /home/frobenius/Desktop/Femtech/HR/App

# Install dependencies (if not already done)
npm install

# Build for production
npm run build
```

This creates the `dist/` folder with:
- Optimized JS/CSS bundles
- PWA manifest.json
- Service worker
- App icons

### Step 2: Upload to cPanel

Upload these files/folders to `hrapp.tripa.com.ng`:

**Required:**
- ✅ `server.js`
- ✅ `package.json`
- ✅ `.env` (or create on server)
- ✅ `dist/` (entire build folder)

**DO NOT upload:**
- ❌ `node_modules/` - Install on server
- ❌ `src/` - Source code not needed
- ❌ `.git/` - Not needed for production

### Step 3: Install Dependencies

Via SSH or cPanel Terminal:

```bash
cd /home/[username]/hrapp.tripa.com.ng
npm install --production
```

This installs express and production dependencies only.

### Step 4: Set Environment Variables

**Option A: Create .env file**
```bash
cd /home/[username]/hrapp.tripa.com.ng
cat > .env << EOF
NODE_ENV=production
VITE_API_BASE_URL=https://hrapi.tripa.com.ng/api
DOMAIN=tripa.com.ng
EOF
```

**Option B: cPanel Environment Variables**
In cPanel Node.js App interface:
1. Click on your application
2. Scroll to **Environment Variables**
3. Add:
   - `NODE_ENV` = `production`
   - `VITE_API_BASE_URL` = `https://hrapi.tripa.com.ng/api`
   - `DOMAIN` = `tripa.com.ng`
4. Click **Save**

### Step 5: Restart Application

In cPanel Node.js App interface:
1. Click **Restart** button
2. Wait 10-15 seconds
3. Check logs for: "✅ Femtech HR PWA is running on port XXXX"

## 🐛 Troubleshooting

### App Won't Start

```bash
# Check logs
cd /home/[username]/hrapp.tripa.com.ng
cat logs/error.log

# Verify Node.js version
node --version  # Should be v20.x

# Check if port is available
lsof -i :[PORT_NUMBER]
```

### 404 on All Pages

- [ ] Verify `server.js` is set as startup file
- [ ] Check `dist/` folder exists
- [ ] Verify `dist/index.html` exists
- [ ] Restart application in cPanel

### PWA Not Working

- [ ] Check manifest.json loads: `https://hrapp.tripa.com.ng/manifest.json`
- [ ] Verify service worker registered (DevTools → Application)
- [ ] Ensure HTTPS is enabled (required for PWA)
- [ ] Check icons exist in `/icons/` folder

### API Calls Failing

- [ ] Check `VITE_API_BASE_URL` in environment
- [ ] Verify backend is running at `https://hrapi.tripa.com.ng`
- [ ] Check CORS configuration on backend
- [ ] Test API directly: `curl https://hrapi.tripa.com.ng/api/system-complete/readiness`

### Build Errors

```bash
# Clear cache and rebuild
rm -rf dist node_modules
npm install
npm run build
```
